# EduConnect BD

Premium online education platform for Bangladesh — built with Next.js, React, TypeScript, Tailwind CSS, Framer Motion, and Lucide React.

## Features

- **Homepage** — Hero, course search, featured courses/teachers, govt services, education results, testimonials, FAQ, contact, newsletter
- **Courses** — Browse, search, filter, and enroll
- **Teachers** — Profiles with qualifications, contact, WhatsApp, and Facebook links
- **Student Dashboard** — Courses, progress, live classes, videos, PDF notes, notices
- **Admin Dashboard** — Students, teachers, courses, enrollments, payments, meetings, notices, settings
- **GovtInfo Portal** — Official Bangladesh government service links
- **Education Results** — Official board and university result portals
- **Dark Mode** — System preference + manual toggle
- **Responsive** — Mobile-first design with glassmorphism and smooth animations

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
npm start
```

## Project Structure

```
app/              # Next.js App Router pages
components/       # UI, layout, home, courses, teachers
lib/              # Types, utils, data, context providers
```

## Tech Stack

- Next.js 16 · React 19 · TypeScript
- Tailwind CSS 4 · Framer Motion · Lucide React
